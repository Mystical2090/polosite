Installation Guide

Follow these steps to get started

  1. Download the latest release from our GitHub.
  2. Download Geode if you haven't already.
  3. Install the mod using the "file picker" in Geode.
  4. Restart the game.
  5. Click the Settings button to use the menu, and enjoy!
